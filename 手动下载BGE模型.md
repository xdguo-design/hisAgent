# 手动下载BGE模型指南

## 问题
国内无法直接从HuggingFace下载模型，需要手动操作。

## 解决方案

### 选项1：使用ModelScope（阿里云的模型库）

ModelScope是阿里云提供的国内模型库，包含BGE系列模型。

**优点**：
- 国内访问速度快
- 无需VPN
- 模型完整

**步骤**：

1. 安装ModelScope：
```bash
pip install modelscope
```

2. 修改代码使用ModelScope

### 选项2：使用网盘下载

从国内网盘下载已打包好的模型文件：
- 百度网盘
- 阿里云盘
- 蓝奏云

### 选项3：使用智谱AI模型（推荐）

直接使用智谱AI的嵌入模型，无需下载：
- 选择嵌入模型：zhipuai-embedding
- 需要智谱AI账户有余额
- 效果同样很好

## 临时建议

**建议先使用智谱AI模型创建知识库**，等有时间再配置本地模型。

本地模型的优势是免费，但配置比较麻烦。智谱AI模型虽然需要付费，但成本很低（每1000个token约0.001元）。

对于你的HIS系统Java代码存储，智谱AI模型完全够用！
